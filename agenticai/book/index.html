Agentic AI: Build, Ship, Portfolio

From zero to production-grade agents — finish with a GitHub portfolio no two readers share.

How to Use This Book

This book is designed so that every reader finishes with a unique, recruiter-ready GitHub portfolio of working agentic AI projects. Here's how:

  1. Read linearly, build incrementally. Each chapter introduces one core concept, explains the mental model, then hands you a runnable project. The concepts build on each other — don't skip ahead.
  2. Pick your domain variant. Every project comes in six real-world flavors: Tech/Software, Healthcare, Finance, Education, E-commerce, and Legal. Choose one primary domain and stick with it — or mix domains across chapters for maximum portfolio diversity.
  3. Ship to GitHub as you go. Each project variant includes a README.md, Dockerfile, CI/CD config, and a CUSTOMIZE.md with five modifications you can make. Push each project as its own repo. By the end, you'll have 15+ production-quality repos.
  4. Use the capstones to integrate. The four capstone projects in Part 5 combine patterns from earlier chapters into full systems. These are your portfolio headliners.
  5. Go deep with exercises. Every chapter ends with three exercises — conceptual (test your mental model), coding (extend the project), and design (architect something new). These are interview-grade problems.
